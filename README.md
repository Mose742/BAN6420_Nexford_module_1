## Project Overview

This project was developed to automate the weekly payroll processing for Highridge Construction Company.

The project includes two implementations:
- Python (Jupyter Notebook)
- R (Jupyter Notebook)

Both programs:
- Dynamically create a list of 400 workers.
- Generate weekly payment slips using a for loop.
- Assign employee levels based on salary and gender.
- Include error handling (Python version).
- Demonstrate the same payroll logic in both Python and R.

## Files Included

1. nexford_assignment.ipynb - Python implementation
2. R_nexford.ipynb - R implementation
3. README.md - Project instructions

## Requirements

Before running the notebooks, ensure the following are installed:

- Python 3.x
- Jupyter Notebook
- R
- IRkernel (for running R notebooks in Jupyter)

## How to Run the Python Notebook

1. Open Jupyter Notebook.
2. Open the file named **nexford_assignment.ipynb**.
3. Run each cell from top to bottom by clicking **Run** or pressing **Shift + Enter**.

## How to Run the R Notebook

1. Open Jupyter Notebook.
2. Open the file named **R_nexford.ipynb**.
3. Ensure the notebook is using the **R kernel**.
4. Run each cell from top to bottom by clicking **Run** or pressing **Shift + Enter**.
